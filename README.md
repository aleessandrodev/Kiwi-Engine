How to use:
To use the editor, install NodeJS:

Windows:
- Go to the official website: https://nodejs.org
- Download the .msi installer
- Run the setup
Alternatively, you can use winget or chocolatey by running:
<code>winget install OpenJS.NodeJS</code>
or
<code>choco install nodejs</code>

Linux:
Depending on your distribution, run:
<code>-sudo apt install nodejs npm</code>

<code>-curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -</code>
<code>sudo apt install -y nodejs</code>

<code>-sudo pacman -S nodejs npm</code>

<code>-sudo dnf install nodejs npm</code>

If you are on Windows, you can open the .bat file. It will automatically start the server and open a page in your default web browser, allowing you to use the engine.
Alternatively, you can run:
<code>cd path to the folder where server.js is located</code>
<code>node server.js</code>
The same commands apply for Linux.
